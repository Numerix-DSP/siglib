PortAudio Example Programs

  

This directory contains the SigLib example programs for use with PortAudio, which can be obtained from http://www.portaudio.com. Yhese examples use V19 of PortAudio and have been tested on Linux (x86 and Raspberry Pi) and Windows.

Source Files

pa_agc.c - Automatic gain control - echos the input to the output and controls the gain.

pa_fft.c - Fast Fourier Transform.

pa_filter.c - IIR Filtering.

Note examples pa_fft.c and pa_filter.c use the analog_io functions provided to simulate using ADCs and DACs under interrupt control. More details are available in the following blogs:

    How To Connect Digital Signal Processing Functions To An ADC And A DAC

    How To Perform An FFT On A Real-Time Data Stream
