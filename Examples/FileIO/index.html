PortAudio Example Programs

  

This folder contains the SigLib file I/O example programs.

Source Files

    bin2wav.c - Command line application to convert from binary format to mono .wav format
    sig2csv.c - Command line application to convert SigLib .sig files to comma separated variable files (.csv). This is useful when importing SigLib data into a spreadsheet.
    sig2wav.c - Command line application to convert SigLib .sig files to .wav files.
    sigplot.c - Command line application to graphically plot SigLib .sig files.
    wav2dat.c - Command line application to convert from mono .wav format to .dat format.
    wav2sig.c - Command line application to convert from mono .wav format to SigLib .sig format.
    wavinfo.c - Command line application to print the .wav file header information.
    wavplot.c - Command line application to graphically plot mono .wav files.

Batch files are included for building with Microsoft and GCC compilers.
