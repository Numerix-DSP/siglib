SigLib p/4 D-QPSK Modem Example Programs

  

The files in this folder provide a example of a complete p/4 D-QPSK modem.

The input data sequence is taken from the file "base.txt" this data is used to modulate the carrier (using dqpskmod.exe) and the data is written to a .wav file before the data is passed to dqpskdem.exe which demodulates the data. The demodulated data is written to the file demodout.dat. Each data burst contains a precoding region of continuous 1s followed by the packet data sequence which is indicated by the synchronization sequence : 1111...01010001111110. The demodulated synchronization sequences and packet data sequences should be the same as base.txt.

Each data sequence in base.txt is transmitted as a separate data sequence which means that the symbol and carrier synchronizers have to be reinitialized on each iteration. These functions also work with continuous data streams.

The application uses V.26B modulation with the following configuration :

Sampling Rate = 48 KHz
Data format = 8 bit
Symbol Rate = 1200 baud
Bit Rate = 2400 bps
Carrier Frequency = 1800 Hz

These eexamples are provided for the Microsoft Visual C/C++ and GCC compilers, using Gnuplot/C which is included in SigLib.

Filename    Description
Base.txt    Source data file that is used by the modulator. Contains pre-amble, data sequence and post-amble
BASE.WAV    Modulated sequence
Demodoutput.txt    Output from demodulator
dqpskdem.c    Demodulator source code
dqpskmod.c    Modulator source code
Rx_FIR.h    FIR filter for demodulator
Tx_FIR.h    FIR filter for modulator

Filename    Description
gBuildRun.bat    Batch file to build and run modulator and demodulator programs with GCC
gb.bat    Batch file to build source files with GCC
mBuildRun.bat
Batch file to build and run modulator and demodulator programs with Microsoft Visual C/C++ compiler
mb.bat 
Batch file to build source files with Microsoft Visual C/C++ compiler

Build example for analyzing the modem functions with gprof are provided in gBuildProfile.bat. Note, gprof is a statistical function with a sampling period of 0.01s. Many of the modem functions take less than this period to run so do not show up in the results.
To execute this under Cygwin, install the following packages :
    Cygwin from https://cygwin.com/install.html and include the following packages : gcc and make
    GraphViz from >http://www.graphviz.org/Download_windows.php and add the bin folder to the path
    Gprof2dot from https://github.com/jrfonseca/gprof2dot. Note, I located the extracted folder off %HOMEPATH%\python
The gprof results are presented in .png call graph images.
