SigLib Machine Learning Example Programs

  

The files in this folder provide examples for the SigLib Artificial Intelligence and Machine Learning functions that run in real-time, utilizing PortAudio to handle the I/O.

This example is tested on a Raspberry Pi, with a Seeed ReSpeaker 2-Mics Pi HAT.
The Pi HAT handles both the analog input and is also used to display the detected vibration mode using the onboard LED.
This code uses the default audio input so will work with any supported Operating System as long as there is a connected microphone.

The model is generated using the example in CExamples/MachingLearning and is provided in the header file weightsFloatMotorMode1_MotorMode2.h

Source Files    Description
pa_ml.c    Performs inferencing detection of current vibration mode.
weightsFloatMotorMode1_MotorMode2.h    Trained model weights file.
rbr.sh    Shell script for building and execuring the code.
analog_io.c    Analog input support file.
mic_hat_led_driver.c    Mic HAT interface file.

Please also see the example in CExamples/MachineLearning for the offline training and inferencing programs.
