SigLib Kalman Filter Example Programs

  

The files in this folder provide examples for the SigLib Kalman Filter functions. The examples include programs for the following:

  • A 1D Kalman Filter to estimate position.
  • A 2D Kalman Filter to estimate position and velocity.
  • A 2D Kalman Filter to implement the planeKF example here: Kalman Filter YouTube Lessons .

These examples are provided for the Microsoft Visual C/C++ and GCC compilers. For graphical result display they use Gnuplot/C, which is included in the SigLib download pachage.

Source Files    Description
kalman_1D.c    A 1D Kalman Filter to estimate position
kalman_2D.c    A 2D Kalman Filter to estimate position and velocity
planeKF.c    A 2D Kalman Filter to implement the planeKF example
 
Build Scripts 
gb.sh    Shell script to build any of the examples with GCC
gbr.sh    Shell script to build and run any of the examples with GCC

Using The Examples

The kalman_1D and kalman_2D examples can be run with graphical ouput using the following commands:

  • ./gbr.sh kalman_1D -g
  • ./gbr.sh kalman_2D -g
