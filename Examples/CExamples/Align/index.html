SigLib Audio (.wav) Alignment Example Program

  

The files in this folder provide examples of how to use the alignment functions.

The input files have been generated from the following reference : https://www.gutenberg.org/ebooks/23967

The batch files supplied work with the GCC and Microsoft Visual Studio compiler (32 and 64 bit versions supplied) however the example programs can be built using almost any 32 or 64 bit compiler.

Filename    Description
alignwavs.c    Audio alignment using cross-correlation.
