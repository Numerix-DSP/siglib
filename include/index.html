SigLib Include Files

  

Return To The Top Level | Source files | Complete include directory

siglib.h - Main SigLib include file, also includes the followin files:
siglib_constants.h - Constant definitions
siglib_types.h - Structure and enumerated type definitions
siglib_macros.h - Macro definitions
siglib_processors.h - Processor dependent definitions and declarations

Compiler Specific Header Files

siglibv.h - Agilent VEE include file, generated from siglib.h using hpveegen.bat.

SIGLIB.PAS - Pascal/Delphi file, generated from siglib.h using DelphiGen.bat.
