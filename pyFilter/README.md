# SystemAnalyzer

            Numerix pyFilter Filter Design Apps

                      Version 1.00

                       18/11/2025


** INTRODUCTION

pyFilter is a collection of filter design programs.

These support both FIR and IIR filters and low-pass, high-pass,
band-pass and notch filters.

This package currently includes:
  - pyFIR.py    - FIR filter design package
  - pyIIR.py    - IIR filter design package

** HISTORY NOTES

    18/11/2025    V1.00     Initial release

** DEPENDENCIES
These Python scripts require the following packages to be installed:
  - configparser
  - Numpy
  - PyQt5
  - PyQtGraph
  - SciPy
